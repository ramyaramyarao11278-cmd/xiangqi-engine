# 贡献指南

感谢你对中国象棋 AI (Xiangqi Engine) 感兴趣！我们非常欢迎任何形式的贡献，包括提交 Bug、改进文档、提出新功能建议或提交代码。

## 如何开始

1. **Fork 本仓库**：点击右上角的 "Fork" 按钮。
2. **克隆代码**：将代码克隆到本地。
   ```bash
   git clone https://github.com/YOUR_USERNAME/xiangqi-engine.git
   ```
3. **创建分支**：
   ```bash
   git checkout -b my-new-feature
   ```

## 开发流程

1. **环境搭建**：请参考 `README.md` 中的开发指南。
2. **代码风格**：请遵循 PEP 8 规范。
3. **提交代码**：
   - 确保代码可以运行。
   - 编写清晰的 Commit Message。
   - 如果是新功能，请确保添加了相应的测试（如有）。

## 提交 Pull Request

1. 将分支推送到你的 GitHub 仓库。
   ```bash
   git push origin my-new-feature
   ```
2. 在原仓库页面点击 "New Pull Request"。
3. 填写详细的描述，解释你的修改内容。

## 报告 Bug

请在 Issues 页面使用 "Bug Report" 模板提交 Bug。请包含：
- 操作系统版本
- Python 版本
- 复现步骤
- 错误日志

## 行为准则

请保持友好和尊重。我们希望创建一个开放、包容的社区。

感谢你的贡献！🚀
